ABOUT THE PROJECT

Our goal is to classify, display, and translate words that are inputted by a user. Using our language detection dataset, provided by Kaggle, we are creating a model which will predict, then classify, those inputted words to a language such as English, Spanish, Arabic, Russian, and Chinese. We strive to create a Natural Language Processing model that performs close to other models such as Google Translate.

Project Created by: Joshua Korang, Donye Wakefield, Jesse Byler, Giovanni Ruggirello, and Alex Berger. Created for Education Purposes. Source code available at Eastern University's Github
